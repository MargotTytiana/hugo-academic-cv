#### Personal Academic Website & Portfolio

This repository contains the source code for my personal academic and professional website. It showcases my academic background, research projects, and technical skills. 

##### 👨‍💻 About

I hold a Master's degree in Signal Processing and Machine Learning from Tampere University, and a Bachelor's degree in Software Engineering. This website serves as a central hub for:
* My academic CV and educational background
* Projects related to Machine Learning, Signal Processing, and Software Engineering
* Contact information and professional links

##### 🛠 Tech Stack

* **Static Site Generator:** [Hugo](https://gohugo.io/)
* **Theme:** Modified from the excellent [Hugo Academic CV](https://github.com/MargotTytiana/hugo-academic-cv) developed by Me. 

#### 🚀 Local Development

To run this website locally on your machine, follow these steps:

1. **Install Hugo:** Ensure you have the [Hugo Extended version](https://gohugo.io/installation/) installed.
2. **Clone the repository:**
   ```bash
   git clone [https://github.com/MargotTytiana/hugo-academic-cv.git](https://github.com/MargotTytiana/hugo-academic-cv.git)
   cd hugo-academic-cv

#### 📦 Deployment

This website is hosted and automatically deployed using [Vercel](https://vercel.com/). 

Every time a change is pushed to the `main` branch, Vercel automatically triggers a new build and updates the live site at:  
👉 **[yueyao-cv.vercel.app](https://yueyao-cv.vercel.app/)**
